  • ¿cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    • GET: obtener paquete
    • HEAD: obtener los headers
    • POST: envio de datos al servidor
    • PUT: envio de datos al servidor para reemplazar paquete
    • PATCH: envio de datos al servidor para modificar paquete
    • DELETE: eliminar paquete
